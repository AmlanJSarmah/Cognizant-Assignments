## Setting up JUnit (Ubuntu & Intellija)
1. Right-click the `src/` Mark Directory as Test Sources Root
2. File → Project Structure → Modules → Dependencies tab → + → Library → From Java...
3. Navigate to `/snap/intellij-idea/98/plugins/junit/lib/junit.jar` and select it
4. `hamcrest-core` and `junit` is installed at root, `.idea/libraries/junit.xml` points to the installed packages
